# Commande-predective-pour-la-gestion-d-un-microgrid
Ce projet traite la gestion énergétique d’un microréseau résidentiel composé d’une maison consommatrice d’électricité, d’une production photovoltaïque, d’une batterie et d’un échange bidirectionnel avec le réseau par une stratégie de commande prédictive (MPC)
